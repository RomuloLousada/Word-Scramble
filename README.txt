# Word Scramble - Node.js

Aplicação que será executada pelo terminal.
Para iniciar a aplicação, é necessário clonar o repositório utilizando o seguinte comando:

### `git clone https://github.com/RomuloLousada/Word-Scramble.git`

Feito isto, abrir o terminal, navegar até o diretório onde o projeto foi clonado e executar
o comando:

### `node app.js`

# Requisitos:

Possuir o node.js instalado na máquina, em uma versão atualizada.
